Name: Aubie's Memory Game (AubieMemory)

Description:
Modelled after the Milton Bradley game, Simon. There are eight buttons: red, blue, green, yellow,
pink, light blue, orange, and purple. When the game begins a button is randomly selected and a tone
is played. Each time the player successfully enters the correct sequence of buttons, a new randomly
chosen button is added to the end of the sequence the player must follow. The game ends when the
player makes a mistake. There are three difficulties: beginner, intermediate, and expert.

Special Instructions: None.
