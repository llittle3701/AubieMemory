Name: Aubie's Memory Game

Description: Modeled after the Milton Bradley game, Simon. There are four buttons: red, blue, green, and yellow.
             When the game begins a button is randomly selected and a tone is played.
             Each time the player successfully presses the correct button a new randomly chosen button
             is added to the end of the sequence the player must follow.
             The game ends when the player makes a mistake.

Special Instructions: None.
